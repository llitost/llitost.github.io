codeforces 617E XOR and Favorite Number (莫队算法）

发表于 2017-07-16 | 分类于 Contest |

http://codeforces.com/problemset/problem/617/E

题意；给了一个数组，要求询问m次，询问给的区间里有多少子区间Xor后等于k.

预处理完前缀先把左区间减1方便后面的运算，题目的运算满足在知道[l,r]的条件下可以O(1)得到[l-1, r]，[l+1, r]，[l, r-1]，[l, r+1]，因此可以用莫队算法，一种分块优化的思想，先记录要查询的区间然后排序，这样每次只需要控制两端端点的移动即可更新答案
求有多少子区间子异或和为k，如[l,r]区间异或为k, 即sum[l-1]^sum[r]=k; sum[r]^k=sum[l-1]；包含区间[l,r]且异或和为k的区间个数为num[sum[l-1]]=num[sum[r]^k],因此我们只需记录num[sum[r]^k]就行。这里需要注意一下左边界，我们统计num[sum[l-1]].
