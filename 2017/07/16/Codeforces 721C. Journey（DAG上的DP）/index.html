Codeforces 721C. Journey（DAG上的DP）

Posted on 2017-07-16 | In Contest

题目链接：http://codeforces.com/problemset/problem/721/C
题意：给一个DAG,问用不超过T的时间从1到n最多可以经过多少个点.要求输出一条路径;
输入第一行n, m, T， n个点，m条边，时间为t。

没想到是dp题。。。首先定义状态dp[i][j]为经过i个城市到达j城市的最少时间。现在限制了时间和旅程终点，即求满足dp[i][n]<=T的最大的i。
假如现在有u到v的单向路径，那么dp[i][v]=min(dp[i][v],dp[i-1][u]+w[u][v]).第一层从1到n，那么第二层则需遍历图中的每一条边。
此外，需要打印路径，所以增加一个pa[i][j]记录经过i个城市到达j城市的前一个城市pa[i][j]，最后回溯打印路径.其他看代码。
