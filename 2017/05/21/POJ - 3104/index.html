POJ - 3104

发表于 2017-05-21 | 分类于 算法 |
字数统计: 518 | 阅读时长 ≈ 2

https://vjudge.net/problem/16416

题意：题目大意：有n件湿的衣服，每件衣服都有相应的湿度，每分钟每件衣服的湿度减1（除了在烘干机里的衣服，这里重点），现在有一个烘干机，烘干机一分钟可以让一件衣服的湿度降低k，问至少要花多少分钟才能使每件衣服的湿度为0

解题思路：贪心的话，每分钟都要使用到烘干机。先排序，枚举时间，如果湿度小于等于时间的话，就不用考虑了，在枚举时间内肯定会干的。如果湿度大于枚举时间的话，就要考虑一下了，该衣服要在给定时间内湿度变为零的话就要满足该式子，设已经过了cnt分钟了，当前这件衣服的湿度为d[i],枚举的时间为mid，那么 (num[i] - cnt) - n * k <= mid - cnt - n ,也就是n ＝(num[i] - cnt) / (k - 1),如果有余数的话，时间还要加一。另外，特殊处理K<=1的情况。
