A 常数串

Time Limit:1000MS Memory Limit:65535K
题型: 编程题 语言: 无限制
描述
某君做一个测试，然后得到很多数据。这些数据是波动的，但是在其中会出现一段连续的稳定的数字，某君定义为常数串。
给你一个含n个整数的数列，a1, …, an，n个整数，若在区间 [l, r] 中, 即区间al , …, ar之间的最大值M 和最小值m ，有M - m ≤ 1，则称al , …, ar这个数列为一个常数串。
现在你的任务是求给定数列中的最大常数串的长度。

输入格式
多组数据（不超过100组）
每组数据的第一行输入正整数N，表示序列元素的个数;
下一行为一个数列，有N个整数，a1，a2，……aN;
（1 ≤ N ≤ 100 000, 1 ≤ ai ≤ 100 000）
输出格式
每组数据输出一个整数代表答案
输入样例
5
1 2 3 3 2

11
5 4 5 5 6 7 8 8 8 7 6
输出样例
4
5
Provider 201430320515

分析：这题若是嵌套循环，绝大可能是超时(不排除有极好的优化)。故应从O(n)的复杂度思考。由题意，常数串中至多有两个元素，且二者之差的绝对值不超过1。这样我们可以遍历一遍数组，每遍历一个元素，用两个变量记录最新的最值，两个变量记录该最值的位置。然后判断是否合法，合法就更新答案，否则就把最值处于前面的往后移一位，这样对应的最值及位置改变了，直到合法为止，左端点移到相应位置更新答案。这里相当于压缩了数据，把区间内重复的的元素忽视了。例 1 2 2 2 2 2 3 3 3 3，第一个合法区间记录了最后一个2的值和位置，此时，往右遍历，遇到3，更新最值，判断不合法，便把记录1的变量往后移一个，此时则记录到了第一个2，再更新答案，依此类推。

描述
晋太元中，有个渔夫不小心进入了桃源乡
而SCAU ACM的大洲，最近也不小心进入了幻想乡
由于想念SCAU里面的妹子，洲神决定逃离幻想乡，于是洲神找到了幻想乡的出口

但是，这个出口被巫女施下了符咒，必须解开符咒，才能离开幻想乡
这份符咒由n个符文和n个结界组成，符文和结界都有一个唯一且不重复的编号，从1-n
每次大洲都能对其中一个符咒进行施法
如果大洲对第i个符文施法，那么编号为i的倍数的结界状态都会发生改变（从开启变为关闭，或从关闭变为开启）
例如，现在有5个符文和结界，大洲对编号为2的符文施法，那么编号为2、4的结界状态会发生改变
初始状态下，所有结界都是打开的，只有当全部结界都关闭后，大洲才能离开幻想乡

当然，对于大洲神来说，要解开这份符咒是非常简单的，因此怎么解开这个谜题的任务就不交给你了
大洲只是想知道，如果他将1到n的n个符文都施法一次
最后会有多少个结界处于关闭状态

输入格式
输入有一行，为一个数n(n<=2147483647)，代表有n个符文和结界
输出格式
输出一个数，代表最后处于关闭状态的结界的个数
输入样例
2
输出样例
1
Provider 201330340113

分析：一个数的最终状态取决于这个数被改变状态的次数，如果这个数有n个因子，那它就被改变n次，由于完全平方数有奇数个因子，非完全平方数有偶数个，所以最后呈关闭状态的都是完全平方数。

给出n（ n >= 2 ）个整数，其中有 2 个数 a , b 只出现过一次 ， 并且 a != b , 其他数都只出现偶数次 ， 问这两个数分别是什么 ?

第二行输入 n 个数 xi ( 0 <= xi < 10^9 ) , 用空格隔开

输入样例
4
7 7 9 8

Provider 201330330210

参考：http://www.cnblogs.com/youxin/p/3349834.html
分析：先设x = a^b，在知道异或的性质后容易求得x，接下来是如何将a和b分开：
设x在二进制下，从右往左第一个1的位置为k，利用(x>>k)&1 == 1可找到k，因为x是a^b的结果，所以x的第k位为1表明a和b在第k位不同，即一个为0，另一个为1。接着遍历所给n个数，对于第i个数ai，如果ai的第k位为0（也可以是1，目的是分开a和b），即(ai>>k)&1==0，就让x异或ai并存于x，最后的x即是其中一个答案。由于时间限制大，代码需要经过优化才不会超时。
代码小优化：利用lowbit()函数，也即是k = -x&x，如果x二进制是0010 1100，那k = 0000 0100，接着用k&a[i]即可。
代码大优化：全改用getchar()输入，嫌麻烦可以scanf(“%s”,s)，都能得到速度提升。by toshiba

分析：我居然忘了中位数的概念。。惭愧。首先题目要求选的食物是连续的，而且是奇数个。如果在一组数里，k要是中位数那么就需要比k大的数的数量和比k小的数的数量是相等的。
可以从k的位置（sit）向左连续（！）地扫，For(i=1;sit-i>0;i++)一直计数有多少个数比k大（a），多少个数比k小（b），并且每到一个位置都存下a-b这个值出现了多少次（可以用map），
（i是奇数和i是偶数又是不同地，因为待会右边的要和左边的组合在一起，题目要求x要是奇数，那么k的右边给出奇数个数，k的左边也要给出奇数个数，这样总数才能凑成奇数，所以可以分成两个map存，分别代表奇数和偶数位置这个差值出现了几次）
而且如果i是偶数并且a==b，answer还要++，这代表连续的从k的位置到这里是一种可以取的情况
然后在 从k的位置向右连续的扫，for(i=1;sit+i<=n;i++)，计数有多少个数比k大（c），多少个数比k小（d），看d-c的值，如果这个差值在刚才扫左边的时候出现了，并且那个位置到这里总共是奇数个数，那就是一种可能的情况，所以看i的奇偶性，i的奇偶性对应的map里，d-c这个值出现的次数就是可以组合在一起的情况（q），所以answer+=q；与左边一样的，如果i等于偶数并且c==b那么answer还要++；最后输出answer。**from bibi

K 有多少个loli number ?
Time Limit:1000MS Memory Limit:65535K
题型: 编程题 语言: 无限制
描述 hl是数论弱弱，经常遇到数论题都是扔给邪神去想。邪神往往瞬间就推出了公式，然后hl才去码。 这次题目意思是给出 n 个数A1 ~ An，n表示用前n个数字（从1开始），每个数字有Ai个，用这 ( A1+A2+….+ An ) 个数能够组成的数字邪神称之为loli number , 现在问题来了，hl想知道这究竟有多少个不同的loli number呢？ 这次hl希望你们能够帮他求出来，免得又得bg邪神了。
输入格式
第一行输入一个数 n（ 0 < n <= 9 ）, 然后下一行输入n个数Ai ( 1 <= i <=n , 0=<Ai<= 12 ) 。
输出格式
输出一行，为不同loli number 的数量Result（ 0 <= Result < 2^63 ）。
输入样例
2
1 1
输出样例
2
Hint
样例中，有1个1 ， 1个2 ， 能组成不同的loli number分别为12、21 。
Provider 201330330210
分析：基础组合题，然而。。。。先统计总共有的元素（不论重复）个数n，先从n中取a1个元素，为C(a1,n)，然后总个数为n-a1，再取a2个元素，为C(a2,n-a1)。依此类推，根据乘法原理，最后答案为相乘的结果。注意，计算组合数时应用long long。

L 又是MC的问题
Time Limit:2000MS Memory Limit:65535K
题型: 编程题 语言: 无限制
描述
众所周知，MC是个世界全是方块的游戏，所以，如果要在这个游戏中设计出曲线的话，就要像像素画一样进行描绘出相近的形状。

2
3
4
5
6
7
8
9
10
输出样例

5
21
37
61
89
129
