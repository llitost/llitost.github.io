向量vector

Posted on 2017-03-02 | In STL

vector是STL中最常用的容器，其特点是占用连续内存空间，对数据的随机访问支持得很好，既可以像普通数组那样用被访问元素的下标来访问，也可以使用向量所提供的成员函数at()来访问。
向量定义于头文件vector，命名于名字空间std。

1.vector对象的定义和初始化

构造函数 作用
vector(); 创建一个没有任何元素的空向量　
vector(vector&& _Right); 创建一个向量，并使用向量_Right初始化该向量
vector(size_type Count); 创建一个大小为Count的向量
vector(size_type Count,const Type& _Val); 创建一个大小为Count的向量，该向量中所有元素初始化为_Val
template vector(InputIterator _First,InputIterator _Last); 创建一个向量，并以迭代器_First和_Last之间的元素初始化该向量

另外，vector重载了运算符operator=，故用户可以赋值的方式在两个vector对象之间复制数据。

2.读取vector数据元素

访问表达式 说明
reference front(); const_reference front() const; 返回第一个元素
reference back(); const_reference back() const; 返回最后一个元素

注意，向量容器中第一个元素的位置是0。

3.向量容器提供的各种用于改变数据的成员函数

成员函数 说明
reference operator; const_reference operator const; 返回由Pos指定位置上的元素
reference at(size_type Pos); const_reference at(size_type Pos) const; 返回由Pos指定位置上的元素
clear() 从容器中删除所有元素
erase(position) 删除由position指定位置上的元素
erase(beg,end) 删除从beg到end-1之间的所有元素
insert(position,elem) 将elem的一个拷贝插入由position指定位置上，并返回新元素的位置
insert(position,n,elem) 将elem的n个拷贝插入由position指定位置上
insert(position,beg,end) 将从beg到end-1之间的所有元素的拷贝插入elem的n个拷贝插入由position指定位置上
push_back(elem) 将elem的一个拷贝插入末尾
pop_back() 删除最后元素
resize(num) 将元素个数改为num，如果size()增加，由默认构造函数负责创建这些新函数
resize(num,elem) 将元素个数改为num，如果size()增加，由默认构造函数将这些新元素初始化为elem

4.描述自身状况的成员函数

表达式 作用　
size_type capacity() const; 返回在当前存储空间下，容器可容纳元素的数目
bool empty() const; 若容器为空，返回true；否则返回false
size_type size() const; 当前向量长度
size_type max_size() const; 返回可以输入到容器元素的最大数目

5.C++11标准新增的部分内容

  • 允许用户使用{}进行初始化和赋值
    初始化： vectorv1={1,2,3}; 或者：vector v1{1,2,3};
  • 元素遍历并进行简单操作
    例for(auto &i : v1) i*=i;遍历元素进行平方运算并返回结果。
    for(auto i : v1)cout<<i<<endl;遍历并输出元素。
    以上只是一部分内容，挑选了比较常用的内容。
