StripePainter - SRM 150 Div 1
动态规划的训练好题
题意：给你一个字母串，每个字母代表一种颜色，去粉刷出这样的串，问至少需要多少步。
首先观看官方题解[https://community.topcoder.com/tc?module=Static&d1=match_editorials&d2=srm150]
状态：记dp[left][len][color]为起始位置为left，长度为len，粉刷了color的基础上最少的粉刷次数。

min[left,size,color] = the minimum number of brushes required to paint position
left, left+1, … , left+size-1 of the original strip,
assuming these position currently has color color.

我们从起始位置开始检查，如果为正确的颜色，就接着计算dp[left+1][len-1][color]，这样就少关注了一个点。否则，我们循环尽可能的长度,如下：

loop i from 1 to size
sum = 1 + min[left+1,i-1,stripe[left]] + min[left+i,size-i,color];
update best solution found so far if necessary
end loop

得到状态转移方程：
int minnum=1+dp[i+1][j-1][left[color]]+dp[i+j][len-j][color];
///代表的意义：在粉刷了color的基础上，再用left[color]粉刷i->i+j-1(所以+1)；
dp[i][len][color]=min(dp[i][len][color],minnum);

最终我们要的答案是：dp[0][n][0].
